# R8X Athlete Rating Platform

This project is a multi-school athlete rating platform inspired by GoHornets. It uses React for the frontend, Node.js/Express for the backend, and MongoDB for data storage. Each school has its own login URL and admin dashboard. The platform supports customizable rating logic, API integrations, and athlete data import/merging.

## Features
- School-specific login and user management
- Ratings module with customizable min/max values
- Admin dashboard for API integrations and rating settings
- Athlete data import and merging from multiple sources
- Support for athlete transfers between schools

## Getting Started
1. Install dependencies for frontend and backend
2. Configure MongoDB connection
3. Start backend and frontend servers

## Folder Structure
- `/client` - React frontend
- `/server` - Node.js/Express backend
- `/db` - Database models and config
- `/docs` - Documentation
